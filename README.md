# Test task Worm

[Try it alive](https://grimwoodent.github.io/neotech-worm-test/)

### Development

Scripts: **JS ES6** & **Babel**  
Styles: **Less**  
Build: **Webpack**

#### Install dependencies

```
npm i
```

#### Build

```
npm run build
```

or
 
```
npm run build-prod
```

#### Test

Used **Mocha** and **Chai** for tests

```
npm run test
```
